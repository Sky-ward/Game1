# 2 周迭代路线 + 调参策略

## 第 1 周：打磨核心闭环
- D1-D2：完善战斗数值模型（技能触发、法宝链、羁绊质变）。
- D3-D4：优化地图与事件池，增加 20+ 事件文本与多结局选项。
- D5：完善商店与合成（组件 → 成品、锁定、降价）。
- D6-D7：完善 UI 与视觉占位，建立新手引导。

## 第 2 周：留存与变现
- D8-D9：接入激励视频与首日礼包，完善复活点位。
- D10：抖音侧边栏复访入口优化（引导跳转文案）。
- D11-D12：平衡羁绊断点、海克斯强度（形成核心构筑）。
- D13：准备提审素材，测试包体与合规。
- D14：上线灰度 + 数据监控。

## 调参策略
- 先锁定 3 条主流构筑（剑修/雷法/丹道），保证胜率 45%-55%。
- 对强度高的海克斯降低权重，对弱势构筑提高保底权重。
- 商店权重随幕推进：Act1 偏 1-2 星，Act3 提升 3 星出现。
- 观察失败节点：若第 2 幕失败率过高，适当提高奖励金币或生命。
